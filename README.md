
# 🔒 Content Access Gate (Web3)

> **A premium, Web3-gated content platform. Built for the rapid development competition using Next.js and Google AI Studio.**

## 🌟 Overview

The Content Access Gate is a decentralized application that allows creators to lock premium content behind a Web3 paywall. Users can unlock access by connecting their Stellar wallet and submitting a secure transaction.

### 💎 Key Features
1. **Web3 Paywall:** Seamlessly lock articles, videos, or digital products.
2. **Secure Payments:** Connect via Stellar wallets to unlock premium content.
3. **Live Web3 Integration:** Fully integrated with the Stellar blockchain using `@creit.tech/stellar-wallets-kit`.
4. **Modern UI/UX:** Built with sleek glassmorphism, smooth animations, and Light/Dark modes.

## 🛠 Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Web3:** Stellar SDK & Stellar Wallets Kit

## 🚀 Getting Started

First, install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to interact with the dApp.
